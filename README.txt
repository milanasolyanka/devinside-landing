ЧЁ ЭТО:

Это проект, сделанный на практику в 6 семестре, 3 курс. Февраль 2024.
Фронт и бэк(пока только фронт). Лэндинг-сайт для Тикшаевского девинсайда. Переделывала готовый сайт на тильде на реакт+тс.


СТРУКТУРА ПРИЛОЖЕНИЯ ПО ПАПКАМ:

-- node_modules не трогать, все сломаешь, плакать будешь
-- public иконка, название проекта, важные штуки
-- src тут весь код
---- assets тут картинки, иконки, шрифты(файлами)
---- components это папка с react-компонентами
------ Divider разделительная полосочка
------ Footer :)
------ Header :)
------ InputForm компонента, которая выводит несколько инпутов. По коду, она обновляет несколько инпутов одной функцией без 100500 useState'ов
------ ServiceCard компонента с карточкой услуги в разделе "наши услуги"
------ TechnologyCard компонента с карточкой технологий в разделе "наш стек технологий"
------ UI тут компоненты типа инпутов, кнопок, которые реюзаются в проекте
---- data папка с данными типа длинных текстов и массивов
---- pages папка с секциями сайта
---- styles файлы scss с цветами и шрифтами


КАК СДЕЛАТЬ, ЧТОБ РАБОТАЛО:

1. Проект с гита подгружаем9
2. Открываем в консоли папку, в которой лежит проект
3. Устанавливаем node_modules консольной командой npm install
4. После установки модулей(может занять время) пишем в консоль npm start
5. Приложение открывается в браузере. Вы восхитительны
